# monsterQuest
Text-based adventure game that makes use of several different objects. The user is the "hero" and will choose how many monsters to fight, up to 4. The user will then have the choice to take action, like sword, fireball, shield, or potion. The battle is turn-based, and the hero can only continue to the move on if the current monster is defeated. Created using Python 3.7.4.
